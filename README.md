Smart Gallery 智能相册

功能特性
 
📂 混合管理模式
 
- 支持直接上传图片/视频
- 支持一键扫描并插入服务器已有图片
- 后台智能标识“本地”图片，方便区分管理
 
⚙️ 智能压缩处理
 
- 可选开启 WebP 自动压缩，节省服务器空间
- 自定义压缩质量（0–100）
- 智能内存管理，大图处理稳定
 
🔒 安全与权限
 
- 支持为相册设置独立访问密码
- 本地图片删除仅移除记录，保护源文件安全
 
🎨 前端体验
 
- 自适应布局，兼容 PC 与移动端
- 自定义每行显示数量
- 支持视频直接在线播放
- 弹窗画廊式浏览体验
 
 
 
环境要求
 
- PHP 版本：7.0+（推荐 7.4+）
- 数据库：MySQL
- PHP 扩展：GD2、PDO_MYSQL
 
 
 
安装使用方法
 
1. 下载插件后，将文件夹重命名为  SmartGallery ，上传至  /usr/plugins/  目录

2. 进入后台 → 插件管理 → 启用 Smart Gallery

3. 进入插件设置，配置显示参数与压缩选项

4. 在后台「控制台 → 相册管理」中创建并管理相册

5. 在文章或页面中使用短代码  [gallery]  调用展示
