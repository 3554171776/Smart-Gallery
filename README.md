插件介绍：
Smart Gallery 智能相册
一款专为 Typecho 1.3.0 设计的现代化相册插件，支持本地图片管理、WebP压缩与私密相册功能。
📦 版本信息
当前版本： 1.0.0
适用平台： Typecho 1.3.0+
作者： 落花雨记
主页： https://www.luohuayu.cn
✨ 功能特性
📂 混合管理模式
支持直接上传图片/视频至相册
支持一键扫描并插入服务器已有图片（本地图片）
后台智能标识"本地"图片，方便区分管理
⚙️ 智能压缩处理
可选开启 WebP 自动压缩，有效节省服务器空间
自定义压缩质量（0-100），平衡画质与体积
智能内存管理，大图处理不报错
🔒 安全与权限
支持为相册设置独立访问密码
本地图片删除时仅移除记录，保护源文件安全
🎨 前端体验
自适应布局，完美兼容 PC 与移动端
自定义每行显示数量
支持视频内容直接播放
弹窗画廊式浏览体验
🛠️ 环境要求
PHP版本： 7.0+（推荐 7.4+）
数据库： MySQL
PHP扩展：
GD2（图片压缩功能必需）
PDO_MYSQL
📝 使用方法
下载插件后，将文件夹重命名为 SmartGallery 上传至 /usr/plugins/ 目录
进入后台 -> 插件管理 -> 启用 "Smart Gallery"
点击插件设置配置显示参数与压缩选项
在后台 "控制台" -> "相册管理" 中创建并管理相册
在文章或页面中使用短代码 [gallery] 或 [gallery id=1] 调用
